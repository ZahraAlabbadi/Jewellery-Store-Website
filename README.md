# Jewellery Store E-Commerce Website

## Overview
An e-commerce web application for browsing and purchasing jewellery products.

## Features
- User authentication
- Product browsing
- Shopping cart
- Order processing
- Admin panel for product management

## Technologies
HTML, CSS, JavaScript, PHP, MySQL

## Database Design
Tables include Users, Products, Orders, and Order Details with relational structure.

## Security
- Password hashing
- Input validation
- Prepared statements to prevent SQL injection

## Challenges
- Managing database relationships
- Creating responsive UI
- Implementing secure login system

## Future Improvements
- Payment gateway integration
- Recommendation system
- Cloud deployment
